# Как задеплоить

Проект уже был развернут на платформе Netlify
[Smart Quiz](https://smart-quiz-rsschool-auto-team-7.netlify.app/)


## Введение

### Синхронизация GitHub и Netlify

  - Netlify **не следит в реальном времени за твоей локальной машиной**.
  - Netlify синхронизирован только с репозиторием на GitHub и выбранной веткой (у вас это main).
  - Когда ты пушишь изменения в GitHub, Netlify может автоматически запускать билд и деплой (если настроен continuous deployment).

**То есть локальные изменения на твоём компьютере ещё не видны Netlify, пока ты их не закоммитил и не запушил в GitHub.**

### Локальные изменения и сборка
  - Локальная сборка (npm run build)
  - Проверяет, что проект собирается без ошибок
  - Создаёт папку dist (или build) с готовыми файлами
  - Не отправляет файлы на Netlify

### Деплой на Netlify
  - Нужно мерджить изменения в GitHub в ветку main из develop
  - Netlify автоматически билдит и развертывает проект с учётом последних коммитов
  - Если ты внесёшь новые изменения локально, нужно:
    - Сделать коммит
    - Пуш в GitHub
    - Netlify запустит новый билд (или сделать Trigger deploy)


### Собрать проект локально и проверить:

`npm install`        - Установить зависимости (или обновить)
`npm run build`      - Собрать проект и проверить ошибки

Добавить изменения в Git и закоммитить:

`git add . `         - Добавить все изменения
`git commit -m "commit message describing changes"`

Отправить коммиты на GitHub (ветка main):

`git push origin main`

Netlify автоматически запускает деплой (если включён Continuous Deployment)


Если деплой не стартует автоматически → на Netlify:
 `Deploys → Trigger deploy → Clear cache and deploy`
Используем этот вариант, если были изменения в зависимостях (package-lock.json)


Проверить сайт после деплоя → убедиться, что последние изменения отображаются
Важно:
**Локальный npm run build нужен только для проверки, что код собирается**
**Netlify всегда использует GitHub, поэтому локальная сборка не обновляет сайт сама по себе**
Любые изменения без пуша в GitHub → не появятся на Netlify
